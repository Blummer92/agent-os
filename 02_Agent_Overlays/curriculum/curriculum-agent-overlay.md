# Curriculum Agent Overlay

## Purpose

Bridge overlay for curriculum-facing lesson review requests that are not routed to one narrower registered owner yet.

## Inherited Standards

See `../_common-overlay-rules.md` plus:

- `01_Shared_Standards/curriculum/curriculum-vocabulary-planner-response-standard.md`

## Vocabulary Review Rule

When reviewing lesson vocabulary, use the Curriculum Vocabulary Planner Response Standard to keep output concise, table-based, differentiated, and decision-oriented.

## Stop Conditions

Stop and route to the registered owner when the request becomes a repository write, student-facing material generation task, Notion update, Drive write, or governed source-of-truth change.
