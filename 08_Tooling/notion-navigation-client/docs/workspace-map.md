# Workspace Map

Compact map of Notion workspace areas the agent repeatedly needs. This is a
navigation reference only. Verify live Notion before readiness, status,
ownership, curriculum, or governed-field decisions.

## Suggested structure

For each major workspace area, keep: area name, purpose, primary page or
database link, common child pages or views, owner/steward when known,
source-of-truth status, and notes on how to find active items.
