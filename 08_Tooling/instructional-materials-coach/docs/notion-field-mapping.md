# Applying a Lesson Record to Notion

Each local lesson-candidate YAML field maps directly to a Lessons Learned
property (see
`01_Shared_Standards/notion/notion-learning-databases.md` for the full
schema this mirrors):

| Local YAML field | Notion property |
|---|---|
| `lesson_learned` | Lesson Learned (title) |
| `owner_agent` | Owner / Agent |
| `what_happened` | What Happened |
| `what_to_do_next_time` | What To Do Next Time |
| `guardrail` | Guardrail |
| `severity` | Severity |
| `learning_type` | Learning Type |
| `area` | Area |
| `applies_to` | Applies To |
| `source_type` | Source Type |
| `source_link` | Source Link |
| `follow_up_needed` | Follow-up Needed? |
| `surface_before_work` | Surface Before Work? |
