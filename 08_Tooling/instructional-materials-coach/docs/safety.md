# Safety Boundaries

This tool only ever creates NEW files by duplicating an approved template
into an explicitly confirmed Drive folder ID. It never edits a template or
master file directly, never writes outside the confirmed target folder,
and never changes sharing or permissions.

See `02_Agent_Overlays/instructional-materials-coach.md` for the full
Owned/Allowed/Blocked Write Surfaces this tool operates under, and
`01_Shared_Standards/google-workspace/drive-docs-sheets-safety.md` for the
underlying Workspace safety rules it follows.
