# Open And Upload This

Follow these steps exactly.

1. Download `agent-os-browser-upload.zip`.
2. Unzip it.
3. Open the extracted `agent-os-browser-upload` folder.
4. Select everything inside that folder.
5. Drag the selected files and folders into GitHub's Upload files page.
6. Do not drag the ZIP.
7. Do not drag the parent folder.
8. Before committing, confirm GitHub's upload list shows:
   - `VERSION.md`
   - `CHANGELOG.md`
   - `00_Governance/`
   - `01_Shared_Standards/`
9. The upload should overwrite the starter `README.md`.
10. Commit with:

```text
Initial Agent OS knowledge base draft
```
