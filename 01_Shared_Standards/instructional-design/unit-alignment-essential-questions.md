# Unit Alignment: 12 Essential Questions

Verify these 12 essential questions after the six canonical alignment checks pass.
These deeper checks ensure units are rigorous, relevant, accessible, equitable,
and student-centered.

## The 12 Essential Questions

| Essential Question | 3 means (Meets) |
|---|---|
| **Essential Questions / Big Ideas** | Unit organized around 1-2 big ideas beyond standards compliance. EQs guide all activities. |
| **Prerequisite Skills** | Assessment confirms students have necessary background. Gaps trigger pre-teaching or scaffolds. |
| **Vocabulary / Academic Language** | Key terms defined at 9th-grade level. Multilingual/content literacy support present. |
| **Cognitive Rigor / Depth of Knowledge** | Tasks require thinking level matching standard (not just topic). Evidence at appropriate DOK. |
| **Student Relevance / Authenticity** | Unit connects to students' interests, identities, media, choices, audience, purpose. |
| **Differentiation / Accessibility** | Multiple access points for learning modalities. Scaffolds, accommodations, UDL present. |
| **Pacing / Sequence** | Realistic timeline given complexity. Learning progression builds logically. Time for revision. |
| **Materials / Resources Alignment** | Readings, slides, examples, videos, tools actively support objectives (not filler). |
| **Feedback Cycles** | Students receive formative feedback and chances to revise before summative assessment. |
| **Equity / Representation** | Examples, media, tasks represent diverse identities. No exclusion/stereotyping. |
| **Transfer / Application** | Tasks require applying skill in new context, not just repeating examples. |
| **Student Voice / Choice** | Students have agency: choice of topic, format, audience, or approach. |

## Scoring Scale

- 1 = Not Met
- 2 = Approaching
- 3 = Meets
- 4 = Exemplary

**Gate rule:** All 12 questions must score ≥3.
