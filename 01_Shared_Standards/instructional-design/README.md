# Instructional Design Standards

Shared standards for agents that generate or review classroom slides,
worksheets, packets, and other student-facing instructional materials.

## Scope

These standards apply when an agent creates, edits, reviews, or routes
student-facing materials for digital media instruction.

## Source of Truth

- Production readiness lives in the owning Notion dashboards.
- Teacher modeling readiness lives with the Modeling Coach workspace.
- Student-facing material quality lives with the Instructional Materials Coach.
- Final material review uses the shared material-quality rubric.

## Files

- `learning-science-rules.md` — core classroom-design rules.
- `production-gates-and-compute.md` — hard-stop gate and compute rules.
- `material-quality-rubric.md` — shared QA contract for slides and worksheets.

## Core Rule

Agents must not generate slides or worksheets until source confidence,
unit readiness, teacher modeling, evidence target, and blocker checks pass.
When a gate fails, route the work instead of drafting a partial product.
