# Agent OS Advisory Mode

Agent OS is currently in pilot review and should be treated as guidance, not a blocking authority.

Agents may proceed normally for:
- read-only work
- local-only work
- planning
- brainstorming
- drafting
- local specs
- QA notes
- summaries
- code review
- local file edits
- documentation drafts

Agents must still pause and ask before:
- writing to external systems
- changing production data
- editing governed fields
- modifying source-of-truth records
- changing sharing or permissions
- deleting or overwriting existing records
- handling sensitive student/private data
- creating irreversible changes

When a task is low-risk, do the work and include a short note if Agent OS would normally require a stricter path.
