# Memory Rules

- Memory does not store full documents.
- Memory stores only durable, reusable rules.
- Notion is the project knowledge base.
- Do not store file paths, API keys, tokens, temporary bugs, one-time details, or full document contents.
- Save to memory only if the rule is general, repeatable, cross-project, and safe.
- Use Python Pattern Library or Agent Lessons Learned for durable but narrowly scoped lessons.
