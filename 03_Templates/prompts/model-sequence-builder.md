# Model Sequence Builder Template

## Purpose

Use this template to build or tighten a teacher modeling sequence so it is clear,
narrow, and directly aligned to what students do next.

## Interface

- display name: model-sequence-builder
- short description: Build a tight teacher model sequence aligned to the task.
- default prompt: Use this template to build or tighten the teacher's modeling
  sequence for this lesson so it is clear, narrow, and directly aligned to what
  students do next.

## Use When

- The model is too broad, jumpy, or hard for students to follow.
- The teacher knows the task but needs the live modeling order.
- The model needs to connect more tightly to the student product.
- Students need a clearer path from watching to trying.

## Do Not Use When

- The main need is exact spoken teacher language; use teacher-talk rehearsal.
- The main need is section-by-section materials audit.
- The first problem is choosing which file or source to use.
- The user wants a full unit alignment review.

## Sequence Checks

A strong model sequence has:

1. a narrow skill focus
2. a visible student-facing goal
3. one worked example or partial example
4. small steps in student practice order
5. teacher think-aloud moments at key decisions
6. one likely error and correction move
7. a handoff into guided or independent practice
