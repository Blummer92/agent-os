# Teacher Talk Rehearsal Template

## Purpose

Use this template when the teacher needs exact classroom-ready language,
think-alouds, prompts, transitions, or rehearsal support for a live teaching
moment.

## Use When

- Teacher language is stiff, overloaded, or hard to say aloud.
- The teacher needs a launch, transition, direction set, think-aloud, prompt, or close.
- The hardest live-teaching moment needs rehearsal.
- Likely confusion needs a stronger response line and teacher move.

## Do Not Use When

- The main request is a section-by-section audit.
- The first problem is choosing which file to use.
- The model sequence itself is missing or misaligned.
- The user wants whole-lesson redesign rather than spoken teacher language.

## Workflow

1. Identify the exact live-teaching moment.
2. Ground the talk in objective, student task, likely confusion, and next step.
3. Choose replacement language, talk track, or rehearsal coaching.
4. Write in natural spoken classroom language.
5. Make thinking visible when modeling reasoning.
6. Pair trouble spots with one response line and one teacher move.
7. Lead with usable wording before explanation.

### Replacement Language

### Talk Track With Moves

### Rehearsal Coaching

- say this:

## Quality Bar

The output should sound speakable, align tightly to what students do next,
improve clarity without over-scripting, and give usable lines instead of only
coaching commentary.
