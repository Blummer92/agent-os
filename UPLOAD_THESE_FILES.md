# Upload These Files

Use this folder for manual GitHub upload:

`/workspace/output/agent-os-manual-upload-final/`

## Upload Steps

1. Open `https://github.com/Blummer92/agent-os`.
2. Click **Add file**.
3. Click **Upload files**.
4. Open `/workspace/output/agent-os-manual-upload-final/`.
5. Drag the **contents inside** `agent-os-manual-upload-final/`.
6. Do **not** drag the folder itself.
7. Overwrite the starter `README.md`.
8. Commit with:

```text
Initial Agent OS knowledge base draft
```

## Important

- Upload the files and folders inside the folder, not the folder wrapper.
- Do not upload any ZIP file.
- Do not add `.env`, credentials, tokens, or local cache files.
